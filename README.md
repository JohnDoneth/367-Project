# CIS 367 Project

Marble maze game where you use a phone's gyroscope sensors to move a marble through a maze.

## Build & Run (yarn)

```shell
# install project dependencies
yarn

# run project w/ hot reloading
yarn serve

# build project for distribution
yarn build
```

## Build & Run (npm)

```shell
# install project dependencies
npm install

# run project w/ hot reloading
npm run serve

# build project for distribution
npm run build
```
